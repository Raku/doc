=begin pod

=TITLE enum NaN

=SUBTITLE Not a Number Floating-point number

    enum NaN is Num { }

C<NaN> is a special L<C<Num>> value that normally results from supplying
nonsensical inputs to a floating-point operation.

Many operations on C<NaN> which are otherwise defined to return
certain values may return C<NaN> instead.

=end pod
